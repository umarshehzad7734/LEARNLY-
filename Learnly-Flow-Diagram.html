🔄 Key End-to-End Flows

📄 Material Indexing Flow

Teacher uploads file → DocumentParser extracts text → AdaptiveChunker splits → HuggingFace embeds → FAISS stores vectors

💬 AI Chat (RAG) Flow

Student asks question → Moderation check → Embed query → FAISS similarity search → Groq LLM + context → answer

📝 Quiz Generation Flow

Teacher selects materials → Fetch material text → Groq generates MCQs (JSON) → Normalize & validate → Store in MongoDB

🎯 Adaptive Quiz Flow

Student opens quiz → Check competency score → Determine difficulty tier → Regenerate personalized Qs → Grade & update score
